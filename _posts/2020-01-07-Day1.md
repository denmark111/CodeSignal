## 2. centuryFromYear
인자로 주어진 연도를 세기로 변환합니다.  
작성한 코드
``` Python
def centuryFromYear(year):
    firstHalf = int(year / 100)
    lastHalf = year % 100
    if lastHalf > 0:
        firstHalf += 1
        
    return firstHalf
```

추천수 많았던 코드
``` Python
def centuryFromYear(year):
    return (year + 99) // 100

```
